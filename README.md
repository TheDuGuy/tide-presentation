# Tide POS CRM Strategy Presentation

## How to Present

1. **Open in browser**: Double-click `index.html` or drag it into Chrome/Safari/Firefox
2. **Navigate slides**:
   - **Arrow keys** (← →) or **Page Up/Down**
   - **Spacebar** to advance
   - **Click** right side of screen to advance, left side to go back
   - **Swipe** on touchscreen devices
   - **Home/End** keys to jump to first/last slide

## How to Export to PDF

### Option 1: Chrome (Recommended)
1. Open the presentation in Chrome
2. Press `Cmd + P` (Mac) or `Ctrl + P` (Windows)
3. Set Destination to "Save as PDF"
4. Set Layout to "Landscape"
5. Set Paper size to "Tabloid" or "A3" for best results
6. Enable "Background graphics" in More settings
7. Click Save

### Option 2: Print to PDF from Safari
1. Open in Safari
2. File → Print
3. Click "PDF" dropdown → "Save as PDF"

## Tips for Presenting

- **Full screen**: Press `F11` (Windows) or `Cmd + Ctrl + F` (Mac in Chrome) for distraction-free presenting
- **Share your screen** on Google Meet and navigate with arrow keys
- The progress bar at the bottom shows your position in the deck
- Slide numbers appear in the bottom-right corner

## Customisation

The presentation uses CSS variables for easy theming. Edit the `:root` section at the top of the `<style>` block to adjust colours:

```css
:root {
    --tide-blue: #1A1F71;      /* Primary brand colour */
    --tide-light-blue: #4A90D9; /* Secondary blue */
    --tide-accent: #00D9A5;     /* Accent/highlight colour */
    ...
}
```
